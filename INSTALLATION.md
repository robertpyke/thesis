Installation Guide
====================

Technologies Used
-------------------

- Ruby 1.9.3
- Rails 3.2.11
- psql (PostgreSQL) 9.1.7
- Git (version control)

Steps
------

1. Install Ubuntu 12.10
2. Update your install
```bash
  sudo apt-get update && sudo apt-get upgrade
```

3. Install Ruby1.9.3
```bash
   sudo apt-get install ruby1.9.3
```

4. Install Git
```bash
  sudo apt-get install git-core
```
